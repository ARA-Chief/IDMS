# IDMS Data Files

This folder contains event logs and reports generated by IDMS during operation.

## Structure

```
data/
  factory/
    logs/       ← Per-user session logs:  report-rgarcia-2026-04-19.json
    reports/    ← Daily aggregated reports: report-factory-2026-04-19.json
  engine/
    logs/
    reports/
  deck/
    logs/
    reports/
```

## Log files
Written automatically by the app. Each file covers one user's events for one date.
Safe to open and read in any text editor. Do not edit manually.

## Report files
Aggregated from all user logs for a given department and date.
These are the files used to generate charts and summaries (Phase 8).

## Retention
Files are not deleted automatically. Archive or remove old files manually as needed.
